이미지출력

img-element

  • img element는 alt attribute를 이용해 접근성과 표준성을 준수해야 한다.
  • img element의 크기는 width, height중 하나만 지정하면 비율(ratio)을 유지할 수 있다. 다만, 아닌 file도 있으므로 해당 file은 비례식으로 구하면 된다.(원본 width, height보다 더 크게 하면 img pixel이 깨져버린다.)
  • img element의 width와 height를 지정해야할 경우 반드시 object-fit을 지정해야함.
  • object-fit : img, video 같은 element의 크기를 어떤 방식으로 조절해 element에 맞출 것인지 지정해준다.
  • object-position : x축, y축순으로 기재한다.(right/center/left top/center/bottom 또는 pixel이나 %로도 지정할수 있다.
  • cover는 여백없이 크기에 맞춰 채워주는것, contain은 크기에 맞춰 img를 줄여서 넣는다.(위아래 letterbox가 생긴다.)

background

background를 이용한 roll-over effect 구현

  • backgound:url(경로);
  • background-image:url;

repeat과 position

  • backbround-image는 pattern형태로 적용된다.
  • background-repeat : no-, -x(x축만 반복), -y(y축만 반복)
  • background-position : x축, y축순으로 기재해야한다.

background-image와 -color같이 적용하기

  • background attribute는 color나 image중 하나만 쓸 수 있다.(따라서 따로따로 쓸것.)

background-size

  • background-size : auto(height를 ratio에 맞춘다.) 100%(width)
  • cover, contain(object와 다르게 letterbox가 생기지 않고 반복된다.)를 기억하자!
  • 따라서 따로 background-repeat을 써줘야한다.
