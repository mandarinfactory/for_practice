form양식과 접근성

  • form : data의 전송 범위
  • legend : form element의 제목(table은 caption)
  • fieldset : form element 안의 내용이나 기능의 범위
  • label : 입력양식에 대한 참조 문구
  • title속성 : 접근성(accessibility)차원에서 사용하는 속성
  • select : label attribute를 option에 추가할 수 있다.
  • textarea : mulit-line 입력양식, element내 문자열을 기본값으로 출력한다.
  • input의 type(다른 element처럼 매우 다양하다.)
    • text
    • password
    • radio + name, value attribute 필수!
    • checkbox
    • email
    • number
    • date
    • time
    • button, submit, reset
  • form element의 다양한 속성
    • placeholder : 미리알려주는 안내문(참조기능)
    • maxlength : 최대 글자 길이를 지정해준다. => input type number에는 허락되지 않는 속성이다.
    • size : 크기를 제어하는 속성 => input type number, textarea에는 허락되지 않는 속성이다.
    • button은 form element내에서 type attribute를 따로 지정하지 않으면 자동으로 submit기능으로 동작한다.
