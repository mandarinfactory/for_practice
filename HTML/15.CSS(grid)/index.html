grid layout

  • grid속성은 parent에 적용, block과 유사하다.
  • grid-child element는 inline, inline-block, block의 속성이 적용되지 않는다.
  • child-element들은 자동적으로 box-sizing이 적용되어서 출력된다.
  • grid컨테이너의 속성
    • grid-template-columns: repeat(갯수, 크기)
    • grid-template-rows : repeat(갯수, 크기)
    • 크기단위 : fr(frame), auto, px, %
  • grid컨테이너의 child 속성
    • grid-column : 시작/끝(=시작-1) => grid-column : 1/4(1 ~ 3)
    • grid-column : 시작/span의 갯수(=시작+1)
    • grid-row : -column과 동일하다.
    • grid-area : 시작줄/시작칸/끝줄-1/끝칸-1
