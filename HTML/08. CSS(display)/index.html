display-level 과 align

display-level

block-level

  • 한줄을 차지하는 element
  • 크기속성을 지정할 수 있음
  • block크기를 조정하더라도 해당 block-level이 임의로 조정되지는 않는다.(여전히 한줄을 차지한다.)
  • h1-6, p, div, ul, li, section등이 있다. => inline/inline-block을 알아두는게 더 빠르다.

inline-level

  • 한 줄로 출력되는 element(따로 줄내림 X)
  • inline element들은 크기를 따로 가질수 없다.(가능은 하다. => margin, padding, inline-block)
  • a, span, input, label, em, b, strong, small등

inline-block level

  • inline과 다르게 크기(block)를 가진다.
  • img, input, button, select, svg, canvas, iframe (이 7개는 알아둘것!)
