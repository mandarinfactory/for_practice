ANIMATION

transition

  • CSS의 속성값이 변하는 시간을 지정한다.
  • 속성값이 숫자일 경우만 가능하다.
  • transition : 속성1 n초(s), 속성2 n초(s)
    속성별로 시간을 다르게 설정할 경우 쓰는 방법이다.
  • transition : all n초(s)
  • transition-delay : 1s,2s,3s...
    활성화-비활성화 순서를 역순으로 정해야한다! => 그래야 자연스럽게 나온다.
  • HTML 요소에 직접적용된 CSS속성값은 화면출력시에 transition이 적용되지 않는다.
    단, JS로 적용시킨다면 새로고침 되더라도 자동적용된다.
  • display : none인 요소는 transition이 적용되지 않는다.(.ex_five)

keyframes, animation

  • 다양한 모션의 구현이 가능하다.
  • que(시작과 끝)간이 자연스럽게 연결되지 않는다. 끝나면 바로 시작지점으로 이동함.
    play상태와 중지상태가 자연스럽게 연결되지 않는다.
