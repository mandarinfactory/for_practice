목록요소

ul

Unordered List(순서가 중요하지 않은 목록)

ul type attribute : circle, disk(default), square등 => li의 기호가 변한다.

내가 좋아하는 음식

  • 김밥
  • 떡볶이
  • 튀김

ol

Ordered List(순서가 존재하는 목록)

type속성 : 1, A, a등

내가 좋아하는 음식의 가격 순위

  1. 튀김 : 5,000원
  2. 떡볶이 : 3,500원
  3. 김밥 : 3,000원
