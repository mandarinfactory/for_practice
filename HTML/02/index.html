HTML & CSS_BASIC

Cascading(순차적으로 내려오는, 폭포수의) Style-Sheet(CSS) => 우선순위가 적용되는 디자인 양식

CSS특징

  • CSS 문법 작성법 : head element내에서 style 요소를 declare후 선택자(속성, 값)형태로 작성
  • inherit가 되는 attribute와 안되는 attribute가 있다.

type선택자

  • tagname : {attribute:value;}
  • decendant 선택자가 우선순위가 높다.
  • class/id는 어떤 element보다 가장 우선순위가 높다.
  • 동일한 element에 동일한 style만 가능하다.

class선택자

  • style element내에서 .classname{attribute:value} declare후,
    html element내에서 class attribute를 호출
  • class는 중복호출, 명명은 가능하지만, id는 한곳만 가능하다.(수정관리가 용이)

id선택자

  • style element내에서 #id{attribute:value} declare후,
    html element내에서 id attribute를 호출
  • id가 class보다 순위가 높다.
  • class와 달리 id는 중복호출, 명명이 불가하다.

inline-style

  • html element의 style 속성을 추가하고 CSS양식으로 작성
  • webpage에 inline-style을 사용하지 않는것이 좋은편, 유지관리가 어렵다.
  • 주로 JS ==> CSS로 이용할때 주로 쓰는편이다.
  • id, class보다 가장 상위의 우선순위를 가진다.
