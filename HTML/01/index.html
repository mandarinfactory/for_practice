HALLO! HTML_BASIC

01. HTML특징과 주의사항

  • HTML은 semantic language이다.
  • element는 tag(link)와 attribute(속성, rel="", href=""등, tag 바로옆에 나온다.)로 이루어진다.
  • element간에는 node가 있다.
    부모(parent), 자식(child), 자손(decendent), 형제(siblings)로 나누어져있다.

02. 중요 semantic elements

  • heading element(h1-h6) : h1은 문서안에 한개 이상 포함해야하는 element이다.
    main, section element안에는 heading element 하나를 포함해야한다.
    heading element안에는 반드시 text가 포함되어야한다.
  • p : paragraph, 문단의 범위
  • main : web-document의 중요 내용의 범위, 한개 이상 포함해야한다.
  • ul(unordered-list <=> ol),li(list)는 목록을 구성한다.
  • em, b, strong : 강조 element
  • a : page의 link를 만드는 element
  • input : web에서 user의 data를 받을수 있는 대화형 control을 생성한다.(bar형 icon을 생성한다.) => button과 같이 많이 사용함
  • div,span : 자유자제로 쓰일수 있는 쉽게 말해 box같은 존재이다. (의미가 없어서 아무렇게나 쓰일수 있다. 레이아웃 같은경우 많이 쓰임.)
