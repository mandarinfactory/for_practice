RESPONSIVE LAYOUT

  • 다양한 device 기기환경에 대응하는 layout => 크기에 상관없이 계속변함
  • 가로 혹은 세로 스크롤바 하나만 사용해야함
  • 공통된 디자인과 크기에 따른 분기별 레이아웃을 구분함
  • 요소별로 반응형 디자인을 완성해나가면서 설계해야함
  • viewport설정(HTML내 head에서 설정) => 웹문서 출력시 해상도를 자동으로 맞춰서 변경해준다.
  • media query : CSS를 이용하여 레이아웃을 분기
  • 분기시점을 정할때 디바이스의 크기는 반드시 피할것!!!!!!!!!
  • 가장 작은 화면에서 정상적인 화면이 출력되어야 함
  • markup(DOM)설계시, 항상! 작은화면을 기준으로 markup해야한다!
