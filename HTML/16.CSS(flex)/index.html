FLEX

  • flex요소는 block과 동일함, 단 children들이 크기가 사라진 block-level 형태로 변한다.

flex-direction

  • grid의 축을 결정(어묵꼬치 / row, row-reverse, column, column-reverse), grid(어묵)
  • reverse일 경우, 객체의 순서와 전체 위치 둘다 바뀐다.(어묵꼬치의 방향도 바뀌지만 어묵간의 위치도 변경된다!)
  • 다시말해, grid의 전체위치와 배열 순서가 뒤집힌다.
  • default는 row로 적용되어 있다.(가로축)

justify-content

  • justify-content는 default로 flex-start가 되어있다.
    • flex-start : 기본값(왼쪽, 윗쪽)
    • flex-end : 기본위치의 반대위치로 보낸다.(axis가 row면 오른쪽으로, column이면 아래쪽으로 옮겨진다.)
    • space-between : children element들의 간격을 자동 분할
    • center : 수평중앙으로 정렬시킨다.

align-items

  • align-items는 stretch가 기본값이다.
    row일경우 grid의 height를 parent의 height와 동일하게 설정하고,
    column일경우에는 grid의 width를 parent의 width와 동일하게 설정한다.
  • center : 수직중앙으로 정렬시킨다.(row일경우 수직중앙, column일경우 수평중앙)
  • flex-start/end : row일경우 위/아래, column일경우 오른쪽/왼쪽으로 정렬시킨다.
