semantic table

  • caption : table element의 제목
  • th : field name을 정의
  • td : data value
  • scope : th, td의 읽는 방향을 정의(column, row)
  • thead : th의 범위(권고요소)
  • tbody : td의 범위(필수요소)
  • tfoot : table의 보조요소
