  • 외부SCSS file에 작성된 변수를 호출할 경우 @import가 선행해야한다.
    => import로 변수를 호출할경우 변수가 변경되었을경우 import를 새로고침으로 저장을 다시 해야 변경된다. => 다시말해, import가 되는 파일을 수정할 경우, import를 하는파일도 수정해야 함.
    (변수는 SCSS변수 대신 CSS변수 사용권장, SCSS변수를 사용할 경우 import없이 해당 파일에서 선언 후 사용) 
                    @import 'SCSS파일명.scss'
  • @mixin()
    여러개의 CSS파일에서 반복해서 사용할 CSS디자인은 @mixin을 사용해서 선언하고, 호출시 @include를 이용한다. parameter(인자)가 사용가능하다. 
                    @mixin btn => 디자인이름(w, h){
                    width : w(100px),
                    height : h(200px), 
                }
                button {
                    @include btn(100px, 200px)
                }
                 => JS function형태랑 비슷하다.
                function foo(a, b){
                    console.log(a + b)
                }
