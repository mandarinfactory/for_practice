• e.target : event 대상 element (안쪽에 있는 element를 우선적으로 찾는다.) e.currentTarget : 정확히 해당 event 대상 element만 찾는다.
  • e.stopPropagation() : 상위요소의 event-bubbling을 중단시켜준다.
  • 인라인스타일 변경
    style.css속성 = '값'
    css속성 중 '-'을 사용하면 다음단어를 대문자로 변경해준다.
    border-radius => borderRadius
  • classList.add('클래스명') classList.remove('클래스명') classList.toggle('클래스명')
  • innerHTML : element 안의 tag와 text를 가져온다.
    '내용' : element안의 tag와 text를 변경한다.
  • getAttribute('속성') : 속성값을 가져온다.
    setAttribute('속성',값) : 속성을 값으로 변경한다.
  • 폼요소.value = form element의 value속성값을 가져온다.
    폼요소.value = 값 : 폼요소의 value속성값을 변경한다.
