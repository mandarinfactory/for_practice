  • 함수선언방식-3가지
    •             function foo () {

            }//1번째 방법 => hoisting이 발생한다.

            foo = function () {
    
            }//2번쩨 방법 => 1번째 방법과 다를게 없다.(전역변수인 window.가 생략된거와 같다.)
            
            const foo = () => {
    
            }//3번째 방법 
            foo(); // 항상 선언 후 호출임을 명시하자!
  • 화살표함수는 인자가 한개일경우, 소괄호(), 또 실행할 code가 return문만 있을 경우 중괄호{}, return을 생략가능하다. 
    •             let fn = parameter => parameter;
            let result = fn(1);
            console.log(result);
            /* 
            arrow일때는 매개변수가 하나면 (), return, {} 생략가능
            원래는 
            let fn = (parameter) => {
                return parameter;
            }
            따라서, => 가 있다면 생략된 함수라고 바로 생각해야한다!
            */
