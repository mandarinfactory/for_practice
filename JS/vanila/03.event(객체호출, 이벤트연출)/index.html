• getElementBy- (CSS선택자를 사용하지 않고 선택)
    getElementByID : 객체를 return해준다.
    getElementByTagName, getElementByClassName : 배열(유사배열)을 return해준다.
    Array.from()을 이용해 배열로 변환 후 반복문(forEach)을 이용해서 event-binding
  • query- (CSS선택자를 이용해서 요소를 선택)
    querySelector : 객체 1개만 return
    querySelectorAll : 배열로 변환 후 return
  • onClick속성을 이용
